The Book Collection

Description:

Keeps a collection of books currently owned filtered by genre. Furthermore, books can be added to a wishlist to be purchased later.

  • Created a SPA using React and Redux for the front end with a Rails API Backend.
  • Integrated react-router along with proper RESTful routing convention.
  • Used redux-thunk middleware to receive data and send data from a server.
  • Implemented redux middleware to respond to and modify state change.
  • Utilized React-Bootstrap to create an appealing user interface.
